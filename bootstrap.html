CSS Toggle Switches

Accessible, screen-reader friendly, flexible, CSS-only toggle switches with full keyboard access and mobile support, using standard form controls.

These examples use Bootstrap, you can also use the toggles alone, or with Foundation.

Download

The toggle switches are very flexible, so you can use a number of classes provided by Bootstrap, to make them look and act exactly as you need them.
